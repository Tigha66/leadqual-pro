# LeadQual Pro - Automated Lead Qualification System

## 💰 What This Is

A complete AI-powered lead qualification system that:
1. **Finds** high-intent prospects automatically
2. **Researches** each prospect (website, tech stack, pain points)
3. **Scores** leads 1-100 based on fit + intent
4. **Books** qualified meetings directly to your calendar
5. **Nurtures** unqualified leads until ready

- **Frontend:** Next.js 14, Tailwind CSS, Shadcn/ui

- **AI:** GPT-4, Tavily API, FireCrawl
- **Integrations:** Calendly, HubSpot, Slack, Gmail
- **Hosting:** Vercel (frontend), Railway (backend)
